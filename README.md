# AI Voice Assistant

## Requirements

- Python 3.12
- Core dependencies:
  - pyttsx3 (alternative to Coqui XTTS v2 for Python 3.12 compatibility)
  - numpy>=1.21.0
  - librosa>=0.9.0
  - soundfile>=0.12.0

## Installation

1. Install Python 3.12
2. Install dependencies:
```bash
pip install -r requirements.txt
```
